jQuery : Write less... do more

1/ jQuery est une API

jQuery est ce qu'on appelle une bibliothèque (library) ou une API (application programming interface) Javascript, c'est à dire un ensemble de fonctions qui permettent d'écrire de manière condensée, les lignes de codes les plus utilisées en Javascript.

La version 1.0 stable de JQuery a été crée en aout 2006 par un jeune homme de 22 ans du nom de John RESIG. Aujourd’hui nous en sommes à : jQuery JavaScript Library v3.6.0

On peut trouver jQuery à cette adresse

jQuery écrit d'une manière plus condensée

Nous allons souvent utiliser les fonctions JavaScript qui permettent d'attraper les balises HTML et les animer, ainsi en JavaScript nous allons écrire d'une certaine manière et en jQuery d'une manière plus condensée.

jQuery est une bibliothèque JavaScript créé afin de faciliter la lecture et l’écriture du JavaScript. Une vingtaine de lignes de code en JavaScript correspond à quelques lignes seulement en jQuery. D’où le slogan : Write less… Do more.

Exemple :

  • JavaScript : document.getElementsByTagName('p')[0].innerHTML = 'Je change le contenu du 1er p de tous les p avec JS.';
  • jQuery : $('p:first').text('Et là je change le texte avec jQuery');

2/ Intégration de jQuery

La bibliothèque jQuery est un fichier JavaScript unique contenant tous ses DOM, événements, effets et fonctions qui peuvent donc être utilisés. On inclut ce fichier dans une page web par l’intermédiaire d’un un lien vers une copie locale ou alors grâce à l’une des nombreuses copies disponibles sur les serveurs publics.

a/ Intégration de jQuery

  • Copie locale, dans le head ou en bas de page ex. : <script src="../jquery/jquery-6.0.js"></script>
  • Copie chez Google : <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  • Il existe deux versions de JQuery, une minifiée et une complète
  • sa balise : $("h4")
  • son #id : $("#mon_id")
  • sa .class : $(".ma_class")
  • par lui-même : $(this)
  • par le type : $("input:text")
  • tous les types : $(":radio")
  • par l'attribut : $("img[alt]")
  • tous les attributs : $("[alt]")
  • par le nom de l'attribut ET sa valeur : $("img[title="un chaton"]
  • sélectionner plusieurs sélecteurs en même temps : $("h1, p, #monID")

3/ Comment sélectionner des éléments ?

  • $('p'); // Je sélectionne tous les paragraphes
  • $('p:first');// Je sélectionne le 1er p trouvé
  • $('p:last');// Je sélectionne le dernier p trouvé
  • $('p[unP]');// Je sélectionne un p avec l'identifiant #unP
  • $('p:eq(1)');// Je sélectionne le deuxième paragraphe trouvé
  • $('p:not(.rouge)');// Je sélectionne tous les p sauf ceux qui ont la classe .rouge
  • $('p.une-rouge');// Je sélectionne tous les p ayant la classe .une-rouge
  • $('#monId');// Je sélectionne un élément avec l'identifiant #monId
  • $('p .unLine');// Je sélectionne tous les éléments ayant la classe .unLine et descendant directement du p
  • $('p > .unLien');// Je sélectionne seulement les éléments ayant la classe .unLien et descendants directement du p
  • $('.unLien + .visite');// Je sélectionne les éléments ayant pour classe .visite précédés immédiatement d'un élément ayant pour classe .unLien
  • $('.unLien ~ .visite');// Je sélectionne les éléments ayant pour classe .visite et précédés ou pas d'un élément ayant pour classe .unLien
  • etc.
  • Voir la liste complète

5/ Les méthodes

  • .css() // ex. $(‘p’).css(‘color’,’blue’); Attention à l'absence de la virgule lors de la dernière propriété ccs apportée.
  • .html()
  • .text()
  • .attr()
  • .addClass
