1/JQuery est une API

JQuery est ce qu'on appelle une bibliothèque ou une API (application programming interface) Javascript, c'est à dire un ensemble de fonctions qui permettent d'écrire de manière condensée, les lignes de codes les plus utilisées en Javascript.

La version 1.0 stable de JQuery a été crée en aout 2006 par un jeune homme de 22 ans du nom de John RESIG. Aujourd’hui nous en sommes à : JQuery JavaScript Library v3.5.1

On peut trouver JQuery (user interface) à cette adresse.

jQuery écrit d'une manière plus condensée

Nous allons souvent utiliser les fonctions Javascript qui permettent d'attraper les balises HTML et les animer, ainsi en Javascript nous allons écrire d'une certaine manière et en JQuery d'une manière plus condensée.

jQuery est une bibliothèque JavaScript créée par John Resig en 2006 afin de faciliter la lecture et l’écriture du JavaScript. Une vingtaine de lignes de code en JavaScript correspond à quelques lignes seulement en jQuery. D’où le slogan : "Write less… Do more" ("on en écrit moi et on en fait plus").

Exemple

  • JavaScript :document.getElementsBtTagName('p')[Ø].innerHTML = 'Je change le contenu du 1er p de tous les p avec JS.';
  • jQuery : $('p:first').text('Et là je change le texte avec jQuery');

2/ Intégration de jQuery

La bibliothèque jQuery est un fichier JavaScript unique contenant tous ses DOM, événements, effets et fonctions qui peuvent donc être utilisés. On inclut ce fichier dans une page web par l’intermédiaire d’un unLien vers une copie locale ou alors grâce à l’une des nombreuses copies disponibles serveurs publics.

a/ Intégration jQuery - méthode d'intégration

  • Copie locale, dans le head ou en bas de page > un exemple : <script src="../jquery/jquery-3.5.1.js"></script>
  • Copie chez Google : <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  • Il existe deux versions une minifiée et une complète
